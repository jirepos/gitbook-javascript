# ES6 객체 리터럴





## 기초

```
let myObj = {
   // 생략
   
}
```

### 프러퍼티

```
let myObj = {
    property1: "SSSS", 
    property2: "MMMM"
}
```

### 함수

기존의 방법

```
let myObj = {
   myFunction = function() {
   }
}

```

메소드 축

```
let myObj { 
    myFunction() {
        // 생략 
    }
}
```
